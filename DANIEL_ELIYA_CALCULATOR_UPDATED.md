## 💰 נוסחת חישוב מדויקת

### 🔧 טופ סרוויס - לפי מדרגות אחוז:

```javascript
אחוז טופ סרוויס = (מכירות בפועל / פוטנציאל מהנציגים) × 100

מדרגות:
• פחות מ-60% → 50 ש"ח לחבילה
• 60%-69%     → 70 ש"ח לחבילה  
• 70%+        → 80 ש"ח לחבילה

עמלת טופ סרוויס = מכירות בפועל × תעריף המדרגה
```

### 🛡️ ביטוח:
```
עמלת ביטוח = ביטוח × 75 ש"ח
```

### 💵 סה"כ עמלות:
```
סה"כ = עמלת טופ סרוויס + עמלת ביטוח
```

**דניאל רואה:**

**החישוב החכם:**
1. **אותה מדרגה** - מחשב תוספת פשוטה
2. **חציית מדרגה** - מחשב **מחדש את הכל** בתעריף החדש!

```

תוצאה:
✅ חצית את המדרגה ל-70%+!
💰 +₪1,480 (כל 37 חבילות ב-80 ש"ח, לא רק ה-5!)
```

### דוגמה 2: אותה מדרגה
```
מצב נוכחי: 35/50 (70%) → 80 ש"ח
אם אמכור עוד 3 → 38/50 (76%) → 80 ש"ח

תוצאה:
💰 +₪240 (3 × 80)
```

## 🧪 דוגמאות בדיקה

### תרחיש 1: מדרגה נמוכה
```
טופ סרוויס: 20
פוטנציאל: 40
ביטוח: 5

חישוב:
• אחוז: 50% → מדרגה "פחות מ-60%" → 50 ש"ח
• טופ סרוויס: 20 × 50 = 1,000 ש"ח
• ביטוח: 5 × 75 = 375 ש"ח
• סה"כ: 1,375 ש"ח

סימולטור: "אם אמכור עוד 5 טופים"
→ 25/40 = 62.5% → חציית מדרגה!
→ כל 25 ב-70 ש"ח = 1,750
→ תוספת: +750 ש"ח (לא +250!)
```

### תרחיש 2: מדרגה גבוהה
```
טופ סרוויס: 35
פוטנציאל: 50
ביטוח: 10

חישוב:
• אחוז: 70% → מדרגה "70%+" → 80 ש"ח
• טופ סרוויס: 35 × 80 = 2,800 ש"ח
• ביטוח: 10 × 75 = 750 ש"ח
• סה"כ: 3,550 ש"ח

סימולטור: "אם אמכור עוד 3 טופים"
→ 38/50 = 76% → אותה מדרגה
→ 3 × 80 = 240 ש"ח
```
