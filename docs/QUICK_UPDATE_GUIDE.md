## 🎯 איך לשנות עמלת מסירות מ-200 ל-400 ש"ח?

### דרך 1: עריכה ישירה (הכי מהיר!) ⏱️ 2 דקות

1. **פתח את הקובץ**
   ```
   js/commission-rules.js
   ```

2. **מצא את השורה** (שורה ~229):
   ```javascript
   'geely-modiin': {
       manager: {
           deliveryRate: 200,  // ← כאן!
   ```

3. **שנה את המספר**:
   ```javascript
   deliveryRate: 400,  // שונה מ-200 ל-400
   ```

4. **שמור** (Ctrl+S)

5. **רענן את הדפדפן** במערכת הראשית:
   - Ctrl+Shift+R (Windows)
   - Cmd+Shift+R (Mac)

6. **בדוק** - הזן נתונים וחשב עמלות

---

### דרך 2: ממשק ויזואלי (נוח יותר) ⏱️ 5 דקות

1. **פתח**
   ```
   commission-manager.html
   ```

2. **בחר את האולם** מהלשוניות למעלה:
   - ג'ילי מודיעין / ג'ילי ירושלים
   - יונדאי אוטולין ירושלים / יונדאי מודיעין
   - מיצובישי מודיעין
   - אומודה ג'אקו מודיעין
   - כלמוביל טרייד אין מודיעין

3. **גלול** ל"מנהל אולם" או "נציגי מכירות"

4. **שדה "עמלת מסירות"**: שנה 200 → 400

5. **לחץ** "💾 שמור והעתק קוד"

6. **פתח** `js/commission-rules.js` (הקוד כבר הועתק!)

7. **הדבק** את הקוד החדש (Ctrl+V)

8. **שמור** (Ctrl+S)

9. **רענן** את הדפדפן (Ctrl+Shift+R)

## 🐛 בעיות?

### "השינוי לא עובד"
✅ רענן עם Ctrl+Shift+R (לא רק F5)
✅ נקה cache: Ctrl+Shift+Delete

### "המערכת לא טוענת"
✅ פתח Console (F12) וחפש שגיאות אדומות
✅ בדוק אם יש פסיק חסר או סוגריים

### "לא בטוח מה לשנות"
✅ פתח את `commission-manager.html` לראות הכל בצורה ויזואלית
