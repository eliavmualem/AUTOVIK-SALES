## 🚀 תרחישי שימוש

### תרחיש 1: מנהל רוצה לעדכן עמלות
```
1. פותח index.html
2. לוחץ על "⚙️ ניהול כללי עמלות"
3. נפתח commission-manager.html
4. בוחר אולם מהלשוניות
5. עורך את המדרגות/הסכומים
6. לוחץ "💾 שמור והעתק קוד"
7. מדביק ב-js/commission-rules.js
8. שומר ורענן
9. ✅ העמלות עודכנו!
```

### תרחיש 2: עובד רוצה לחשב עמלות
```
1. מקבל קישור אישי מהמנהל
2. פותח calculator.html?showroom=XXX&employee=YYY
3. הממשק נטען עם הפרטים שלו
4. מזין נתונים ורואה חישוב בזמן אמת
5. ✅ יודע בדיוק מה העמלה שלו!
```

### תרחיש 3: מנהל רוצה לשלוח קישורים לעובדים
```
1. פותח index.html
2. לוחץ על "🔗 קישורים לעובדים"
3. נפתח employee-links.html עם כל העובדים
4. לוחץ "📋 העתק קישור" ליד העובד
5. שולח לעובד בוואטסאפ/מייל
6. ✅ העובד מקבל קישור אישי!
```
