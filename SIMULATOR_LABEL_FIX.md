## ✅ הפתרון

### 1. שינוי ה-HTML הסטטי
**לפני**:
```html
<div class="simulator-input">
    <span>אם אמכור עוד</span>
    <input type="number" id="extra-deliveries" value="0" min="0" max="50">
    <span>רכבים</span> ❌
</div>
```

**אחרי**:
```html
<div class="simulator-input">
    <span>אם אמכור עוד</span>
    <input type="number" id="extra-deliveries" value="0" min="0" max="50">
    <span id="simulator-item-label">פריטים</span> ✅
</div>
```

**שינוי**: הוספנו `id="simulator-item-label"` כדי לאפשר עדכון דינמי.

### 2. עדכון ב-JavaScript
```javascript
// Update the item label (רכבים/טופים)
const simulatorItemLabel = document.getElementById('simulator-item-label');
if (employeeType === 'operations') {
    simulatorItemLabel.textContent = 'חבילות טופ סרוויס'; ✅
    extraDeliveriesInput.placeholder = 'הזן מספר';
} else {
    simulatorItemLabel.textContent = 'רכבים';
    extraDeliveriesInput.placeholder = 'הזן מספר';
}
```

### 3. עדכון ההודעה הראשונית
```javascript
// Set initial message
if (employeeType === 'operations') {
    simulatorResult.textContent = 'הזן מספר חבילות נוסף'; ✅
} else {
    simulatorResult.textContent = 'הזן מספר רכבים נוסף';
}
```
