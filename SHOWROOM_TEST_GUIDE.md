# 🧪 מדריך בדיקת שמות עובדים

## 🎯 איך לבדוק שהשמות מופיעים

### שלב 1: פתח את index.html
פתח את הקובץ `index.html` בדפדפן.

### שלב 2: עבור לאולם
לחץ על אחד מהאולמות:
- **יונדאי מודיעין** (הטאב השני)
- **מיצובישי מודיעין** (הטאב השלישי)
- **אומודה ג'אקו מודיעין** (הטאב הרביעי)

### שלב 3: מה אמור להופיע?

#### 📊 חלק עליון - הזנת נתונים

תראה **4 כרטיסים צבעוניים** (או פחות, תלוי באולם):

##### יונדאי מודיעין:
1. 👔 **גיא הררי** - מנהל אולם (כרטיס אדום)
2. 💼 **דריו נוטריקה** - איש מכירות (כרטיס טורקיז)
3. 💼 **איש מכירות 2** - איש מכירות (פתוח) (כרטיס טורקיז)
4. ⚙️ **אשר עמר** - איש תפעול (כרטיס כתום)

##### מיצובישי מודיעין:
1. 👔 **שחף כהן** - מנהל אולם (כרטיס אדום)

##### אומודה ג'אקו מודיעין:
1. 👔 **הילה שלומי** - מנהלת אולם (כרטיס אדום)
2. 💼 **שיר עלימה** - איש מכירות (כרטיס טורקיז)
3. 💼 **יהב בן דוד** - איש מכירות (כרטיס טורקיז)
4. ⚙️ **אשר עמר** - איש תפעול (כרטיס כתום)

#### 📊 חלק תחתון - תוצאות

תראה **כרטיסי תוצאות** עם אותם שמות!

## 🐛 אם השמות לא מופיעים

### בדיקה 1: פתח את הקונסול
לחץ `F12` בדפדפן ובחר **Console**.

האם יש הודעות שגיאה? (טקסט אדום)

### בדיקה 2: בדוק שהאולם נטען
חפש בקונסול את ההודעה:
```
✅ Showroom hyundai-modiin initialized successfully
```

אם זה לא מופיע - יש בעיה באתחול.

### בדיקה 3: בדוק שהחודש מוצג
חפש בקונסול:
```
Adding event listeners to 29 inputs for month: 2026-01
```

אם זה לא מופיע - החודש לא נוצר.

### בדיקה 4: בדוק את ה-HTML
בקונסול, הקלד:
```javascript
document.querySelectorAll('.employee-card-title').length
```

התוצאה צריכה להיות **4** (ליונדאי מודיעין ואומודה) או **1** (למיצובישי).

אם זה **0** - הכרטיסים לא נוצרו!

### בדיקה 5: בדוק נראות
בקונסול, הקלד:
```javascript
document.querySelectorAll('.month-content[style*="display: block"]').length
```

התוצאה צריכה להיות **1** (חודש אחד מוצג).

אם זה **0** - החודש מוסתר!

## ✅ מה תוקן

1. ✅ תוקן `data-month-content` במקום `data-month-id`
2. ✅ נוסף `display: block` לחודש הפעיל
3. ✅ נוסף `display: none` לחודשים הלא פעילים
