## ✨ מה נוסף?

### 1. 🎯 סימולציה יעד אישי (לכל נציג)

**מיקום:** בכרטיס ההזנה של דניאל ואלפרד

**שדות:**
- יעד אישי (input)
- מסירות בפועל (input)
- **תיבת סימולציה בזמן אמת** ← חדש!

**מה קורה בזמן אמת:**
- מחשב אחוז עמידה ביעד
- מציג אם עמד ב-100% / 110%
- **מראה כמה בונוס יקבל!**
- מראה כמה רכבים חסרים לבונוס

**דוגמה:**
```
🎯 סטטוס יעד אישי:
✅ 120% מהיעד (12/10)

🎉 בונוס 100%: +240 ש"ח
🎉 בונוס 110%: +120 ש"ח
💰 סה"כ בונוס: +360 ש"ח
```

**צבעים:**
- 🟢 ירוק: 100%+
- 🟡 צהוב: 80-99%
- 🔴 אדום: מתחת ל-80%

---

### 2. 🕵️ סימולציה Mystery Shopper (לכל נציג)

**מיקום:** בכרטיס ההזנה של דניאל ואלפרד

**שדה:**
- ציון לקוח סמוי 0-100 (input)
- **תיבת סטטוס בזמן אמת** ← חדש!

**מה קורה בזמן אמת:**
- מראה אם יש קיזוז (-20%)
- מראה אם יש בונוס (+20%)
- מראה אם אין שינוי (80-89)

**דוגמאות:**

**ציון 75:**
```
⚠️ קיזוז 20% מעמלת הרכבים
ציון 75 מתחת ל-80
```
רקע: אדום בהיר

**ציון 85:**
```
✅ ללא שינוי בעמלות
ציון 85 (80-89)
```
רקע: כחול בהיר

**ציון 95:**
```
🎉 בונוס 20% על עמלת הרכבים!
ציון 95 מעל 90
```
רקע: ירוק בהיר

---

### 3. 🏆 סימולציה יעד סניף

**מיקום:** בראש הדף - תיבת "יעדי הסניף לחודש"

**שדות:**
- יעד סניף (input) ← כבר היה
- סה"כ מסירות בפועל (readonly, אוטומטי) ← כבר היה
- **badge סטטוס משודרג** ← עודכן!
- **תיבת הסבר על בונוס סניף** ← חדש!

**מה קורה בזמן אמת:**
- מחשב אחוז עמידה ביעד
- מראה אם הסניף עמד ביעד
- **מסביר שכל הנציגים יקבלו בונוס!**
- מראה כמה רכבים חסרים

**דוגמה - עמד ביעד:**
```
✅ עמד ביעד! (105%)
כל הנציגים יקבלו בונוס סניף
```
רקע: ירוק

**דוגמה - לא עמד:**
```
⚠️ טרם עמד ביעד (87%)
עוד 4 רכבים ליעד
```
רקע: כתום

**תיבת הסבר:**
```
💡 בונוס עמידה ביעד סניף:
אם הסניף עומד ביעד החודשי, כל נציג מכירות 
מקבל בונוס של 10 ש"ח × סה"כ רכבי הסניף
```
